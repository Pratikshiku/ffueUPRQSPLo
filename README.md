### 一、相关技术
- 后端：`Java`、`JavaWeb` / `Springboot`。
- 前端：`Vue`、`HTML` / `CSS` / `Javascript` 等。
- 数据库：`MySQL`

### 二、相关软件（列出的软件其一均可运行）
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- 等

### 三、功能描述
系统分为（身份）：普通用户（学生）、教师、管理员。

**普通用户功能：**
1. 登录
2. 注册
3. 首页
4. 讨论
5. 公告信息
6. 课程
7. 试卷列表
8. 个人中心
9. 考试记录
10. 错题表
11. 后台管理

**教师功能：**
1. 登录
2. 修改密码
3. 个人信息
4. 讨论管理
5. 公告信息管理
6. 课程管理
7. 课程视频管理、课程视频留言管理
8. 学生管理
9. 作业管理（作业管理、回答管理）
10. 试卷管理
11. 试题管理
12. 考试管理（考试记录、错题本）



**管理员功能：**
1. 登录
2. 个人中心（修改密码、个人信息）
3. 基础数据管理（班级类型管理、讨论类型管理、公告类型管理、课程类型管理）
4. 讨论管理
5. 公告信息管理
6. 课程管理
7. 课程视频管理、课程视频留言管理
8. 老师管理学生管理
9. 作业管理、回答管理
10. 试卷管理
11. 试题管理
12. 考试管理（考试记录、错题本）
13. 轮播图信息
